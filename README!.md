# LNL Group - Modern Professional Services Platform

## Overview
LNL Group is a high-end professional services website designed with a "Tech Minimal" aesthetic. It serves as the digital headquarters for three distinct divisions:
1.  **LNL Group:** The strategic umbrella for enterprise innovation.
2.  **LNL Creatives:** Hyper-local content and brand growth strategies.
3.  **LNL Automations:** Next-gen AI workflows and process orchestration.

## Market Focus: The Raleigh / Durham Flagship
Raleigh / Durham serves as the flagship market for LNL Creatives, establishing the "Local Authority" model that is replicated across other territories.
-   **Theme:** High-Growth Tech Meets Historic Charm.
-   **Primary Neighborhoods:** Glenwood South, North Hills, and Downtown Durham.
-   **Strategy:** Capturing the evolution of the Triangle by anchoring brands in both the modern skyline (The Dillon) and historic staples (Oakwood, American Tobacco Campus).
-   **2025 Landmarks:** The Dillon, Boylan Bridge, State Capitol Holiday Inn, Carol Kare House.

## Tech Stack
-   **Frontend:** React (TypeScript), Vite
-   **Styling:** Tailwind CSS, Shadcn/UI, Framer Motion
-   **Routing:** Wouter (Client-side routing)
-   **State Management:** React Query
-   **Theme:** Next-themes (Dark/Light mode)

## Key Features
-   **Dynamic Market Localization:** The Creatives page adapts its content, pricing, and case studies based on the selected market (Raleigh/Durham, Columbus, Moscow).
-   **Interactive Portfolio:** "Local Authority" gallery with city-specific geotags and filtering.
-   **Tech-Forward Design:** Glassmorphism, subtle gradients, and high-performance animations.
-   **Responsive & Accessible:** Fully responsive layout with dark mode support.

## Project Structure
-   `client/src/pages`: Main page components (Home, Creatives).
-   `client/src/components`: Reusable UI components (Hero, Navbar, Services, etc.).
-   `client/src/data`: JSON data for dynamic content (locations.json).
-   `client/src/lib`: Utilities and helpers.

## Getting Started
1.  Install dependencies: `npm install`
2.  Start development server: `npm run dev`
3.  Build for production: `npm run build`
